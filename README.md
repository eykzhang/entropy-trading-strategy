# Money-Bot

Money-Bot is an algorithmic trading bot written in Python that aims to automate short-term trading strategies. The bot is designed to automatically invest and reinvest profits based on market conditions.

## Goals

- Trade short-term assets (stocks)
- Automatically reinvest profits
- Develop and test simple trading strategies
- Learn basic algorithmic trading concepts

## Timeline

This project is scheduled for Summer 2025 with a deadline of **August 8**. Development will proceed in weekly phases.
